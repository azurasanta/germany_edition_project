Hackathon zum Open Data Day

Am 21.Februar 2015 ist Open Data Day - weltweit. Code for München und die Stadt München veranstalten gemeinsam einen Hackathon. Das steht auf dem Programm:

  • Keynote: "Daten sind das Betriebssystem unserer Gesellschaft" von Dirk von Gehlen, Leiter der Abteilung Social Media/Innovation bei der Süddeutschen Zeitung
  • Offizieller Start des Open-Data-Portals der Landeshauptstadt München
  • Open-Data-Projekte programmieren. Wir laden Euch ein, an einem von drei Projekten mitzuarbeiten:
  • Finder-App (wo finde ich einen Spielplatz, Bankautomat etc.)
  • München Transparent: wie funktioniert Stadtpolitik?
  • Meta-Suchmaschine für Wohnungen in München
Wir wollen Hackathons auch für Fortbildung nutzen: Deswegen gibt es eine Einführung in die Datenvisualisierung, die sich vor allem an Journalisten richtet.

Ihr habt Lust, dabei zu sein? Dann meldet Euch kostenlos auf unser MeetUp-Seite an. Gerne könnt Ihr mit diesem Flyer Werbung für den Hackathon machen. Zum ersten Code for München-Hackathon am 29. November 2014 gibt es einen Rückblick auf Storify.
