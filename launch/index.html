Einladung zum Launch Event

Nach dem erfolgreichen Start von Code for Germany mit 12 aktiven OK Labs in ganz Deutschland, ist es an der Zeit das Programm auch offiziell zu launchen und die ersten Erfolge in den Städten gemeinsam zu feiern. Für einen solchen Launch gibt es keinen besseren Zeitpunkt als rund um das Open Knowledge Festival zu dem auch alle unseren internationalen Partner von Code for America, Laboratorio para la Ciudad aus Mexiko, Code for the Carribean und viele mehr zu Gast sein werden.

Wir möchten Sie/Euch hiermit recht herzlich zum Code for Germany Launch Event einladen!

Wann: Montag, den 14. Juli 2014 | von 18.00 Uhr bis 20.00 Uhr
Wo:GLS Sprachenzentrum | Kastanienallee 82 in Berlin Mitte
