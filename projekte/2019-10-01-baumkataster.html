--- layout: project lab: OK Lab Karlsruhe imgname: karlsruhe/trees.png title: Baumkataster showcase: 1 status: Laufend collaborators: - name: Andreas Kugel - name: Simon Haas links: - name: Karte url: https://codeforkarlsruhe.github.io/baumkataster/ - name: Github url: https://github.com/CodeforKarlsruhe/baumkataster --- Das Baumkatatster der Stadt Karslruhe enthält die Geokoordinaten und Art aller städtischen Bäume. Die Karte zeigt die Positionen auf einer Karte, in der ebenfalls für alle Stadtteile Einwohnerzahl und Fläche angezeigt werden. Für jede Baumart können weitere Informationen in einem Popup dargestellt werden. Zur Auswertung der Daten wird eine Vorverarbeitung mit Python gemacht.
